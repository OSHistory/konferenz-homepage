MapFish

Ein OSGeo WebGIS 2.0 Framework

MapFish ist ein Framework, welches die Bildung von individuell gestalteten und erweiterbaren Web-GIS Anwendungen vereinfacht.

MapFish ist so gestaltet, dass es einfach zu benutzen ist, entweder als selbständige Anwendung oder als Bestandteil einer bestehenden Webseite. Als selbständige Anwendung ist MapFish via einige Parameter konfigurierbar und bietet schnell ein funktionales Web-GIS. MapFish kann auch in bestehenden Webseiten wie CMS oder Informations System orientierte Anwendungen ohne Aufwand integriert werden.

Der Client Teil ermöglicht es via OpenLayers2, ExtJS3 und GeoExt4, weiterführende WEB2.0 Funktionalitäten einzubauen.

Der Server Teil stellt in verschiedene Programmiersprachen flexible Prozesse zur Verfügung (Routing, Suchmaschine, thematische Kartografie, usw).
