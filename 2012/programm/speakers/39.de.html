FOSSGIS2012 - 0.29 05.04.2012

FOSSGIS 2012
Konferenz für freie und Open Source-Software im GIS-Bereich und für freie Geodaten

Veranstaltungen
SOS-Importer

Simon Jirka

Simon Jirka ist der Sensor Web Community Leader der Open Source Initiative 52° North. Er besitzt einen Abschluss als Diplom Geoinformatiker und arbeitet momentan an seiner Dissertation. Darin beschäftigt er sich mit Katalogen für Sensor Netzwerke und das Sensor Web. Arbeitsschwerpunkt von Simon Jirka das Thema OGC Sensor Web Enablement. In diesem Kontext besitzt er ein großes Spektrum an Erfahrung aus internationalen Forschungsprojekten sowie praktischen Kundenprojekten. Darüber hinaus ist Simon Jirka Mitglied der Sensor Web Enablement Arbeitsgruppe des Open Geospatial Consortiums (OGC).
