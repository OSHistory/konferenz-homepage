Workshop: QGIS 3 Workshop

Einführung in die neuen Feature

Referenten

 Klaus Mithöfer
Simon Haufe

QGIS hat den Generationenwechsel von Version 2 auf 3 geschafft. Es gibt viele neue Feature und Möglichkeiten. Der Workshop bietet dazu einen praktischen Überblick.

Neben dem praktischen Ausprobieren der neuen Feature geht es vor allem um den Umstieg von QGIS 2 auf QGIS 3. An Beispielen wird gezeigt, worauf man achten muss, wenn man vorher mit QGIS 2 gearbeitet hat. Welche Funktionalität gibt es nicht, welche wurde wie ersetzt.

QGIS hat den Generationenwechsel von Version 2 auf 3 geschafft. Es gibt viele neue Feature und Möglichkeiten. Der Workshop bietet dazu einen praktischen Überblick.
Dabei geht es neben dem praktischen Ausprobieren der neuen Feature vor allem um den Umstieg von QGIS 2 auf QGIS 3. An Beispielen wird gezeigt, worauf man achten muss, wenn man vorher mit QGIS 2 gearbeitet hat. Welche Funktionalität gibt es nicht, welche wurde wie ersetzt. Grundlegende Themen sind:

* Praktischer Überblick zu QGIS 3
* Wie ist der Stand der Entwicklung. Kann ich oder sollte ich bereits migrieren.
* Was hat sich geändert, wenn man es mit QGIS 2 vergleicht
* Worauf muss ich achten, wenn ich vorher mit QGIS 2 gearbeitet habe
